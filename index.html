TUGAS DDL DAN DML

Perintah DDL

No Perintah Pengertian Syntax Contoh
1 CREATE Perintah yang digunakan untuk membuat , termasuk membuat database baru, tabel baru , dan kolom baru CREATE DATABASE nama_database; CREATE DATABASE mahasiswa;
2 DROP Menghapus database atau tabel DROP DATABASE nama_database; DROP DATABASE mahasiswa;
3 RENAME Mengubah nama tabel RENAME TABLE Jurnal TO Jurnalku; ALTER table mahasiswa RENAME wisudawan;
4 ALTER Mengubah struktur tabel yang telah dibuat ALTER TABLE table_name ADD column_name datatype; ALTER table mahasiswa DROP nama;

Perintah DML

No Perintah Pengertian Sintaks Contoh
1 UPDATE Memperbarui data lama menjadi data terkini UPDATE nama_tabel mahasiswa set nim='L200170107'=L200170113;
2 INSERT Menyisipkan atau memasukkan data baru kedalam tabel INSERT into nama_tabel values (‘value1’, ‘value2’,’ …’) INSERT into mahasiswa values('L200170107','Nur Aini');
3 SELECT Mengambil data atau menampilkan data dari satu tabel atau beberapa tabel dalam relasi SELECT nama_kolom from nama tabel SELECT nama_mhs from mahasiswa where nilai = 70;
4 DELETE Menghapus data dari tabel DELETE nama_kolom FROM nama tabel where kondisi DELETE from mahasiswa
